# Desafio de Programação Orientada a Objetos
Projeto de exemplo desenvolvido como exercício de Programação Orientada a Objetos (POO) no curso Formação .NET Developer.

## Descrição
Este projeto demonstra conceitos de POO em C#: criação de classes, herança, encapsulamento e uso de objetos simples. Contém uma pequena aplicação de demonstração com modelos de `Smartphone` (e suas variações) e um `Program.cs` para execução.

## Pré-requisitos
- .NET SDK (6.0, 10.0 ou superior) instalado

## Como executar
Abra um terminal na pasta do projeto e execute:

```bash
cd "Desafio de projeto-Programação orientada a objetos"
dotnet build
dotnet run
```

## Estrutura do projeto
- `Program.cs` — ponto de entrada da aplicação
- `Models/` — contém as classes do exercício (`Smartphone.cs`, `Iphone.cs`, `Nokia.cs`)
- `Imagens/` — recursos de imagem usados pelo projeto
